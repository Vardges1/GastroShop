### Описание проекта

GastroShop — полнофункциональная e-commerce платформа для продажи премиальных европейских сыров и деликатесов с AI-ассистентом для персонализированных рекомендаций и интерактивной картой регионов происхождения продуктов.

### Технологический стек

**Frontend:**
- Next.js 15 (App Router, SSR, SSG)
- TypeScript
- Tailwind CSS
- Zustand (state management)
- TanStack Query (data fetching)
- Framer Motion (анимации)
- MapLibre GL JS (интерактивные карты)
- React Hook Form + Zod (валидация форм)
- next-intl (интернационализация RU/EN)
- shadcn/ui (UI компоненты)

**Backend:**
- Go 1.21+ (Gin framework)
- PostgreSQL 15 (JSONB, миграции)
- JWT аутентификация (access + refresh tokens)
- bcrypt (хеширование паролей)
- Prometheus (метрики и мониторинг)
- golang-migrate (версионирование БД)

**Инфраструктура:**
- Docker & Docker Compose
- Nginx (reverse proxy, load balancing)
- Multi-stage Docker builds
- Health checks и volume persistence

**Интеграции:**
- ЮKassa (платежная система)
- CloudPayments (альтернативный провайдер)
- Webhook обработка для платежей

### Реализованный функционал

**Для пользователей:**
- Каталог товаров с фильтрацией, поиском и пагинацией
- Корзина покупок с сохранением состояния
- Система заказов с отслеживанием статусов
- AI-ассистент для рекомендаций продуктов на основе естественного языка (например, "сыр к ризотто с грибами")
- Интерактивная карта Европы с отображением регионов происхождения продуктов
- Мультиязычность (русский/английский)
- Мультивалютность (RUB, USD, EUR) с конвертацией
- Личный кабинет пользователя
- Избранное (favorites)
- Адаптивный дизайн (mobile-first)

**Для администраторов:**
- Админ-панель с дашбордом и аналитикой
- CRUD операции для товаров (создание, редактирование, удаление)
- Управление заказами (изменение статусов, просмотр деталей)
- Управление пользователями (роли, блокировка)
- Статистика продаж с графиками (Recharts)
- Система метрик (Prometheus)

**Технические особенности:**
- RESTful API с четкой архитектурой (handlers, services, repositories)
- Middleware для аутентификации и авторизации
- Обработка платежей через webhooks с валидацией HMAC-SHA256
- Идемпотентность операций
- Event tracking для аналитики
- Оптимизация производительности (мемоизация, debounce)
- SEO оптимизация (sitemap, robots.txt, мета-теги)

### Архитектурные решения

- **Монолитная архитектура** с разделением на слои (presentation, business logic, data access)
- **Repository pattern** для работы с БД
- **Service layer** для бизнес-логики
- **Middleware chain** для обработки запросов
- **Docker Compose** для оркестрации сервисов
- **Миграции БД** для версионирования схемы

### Ключевые достижения

- Реализована полная интеграция платежных систем с обработкой webhooks
- Создан AI-ассистент на основе rule-based системы с возможностью расширения до ML-решений
- Разработана интерактивная карта с использованием векторной графики (MapLibre GL)
- Обеспечена типобезопасность на всех уровнях (TypeScript + Go)
- Настроена инфраструктура для разработки и деплоя (Docker, Nginx)
- Реализована система мониторинга и метрик (Prometheus)

### Метрики проекта

- **Frontend:** ~15+ страниц, 30+ компонентов
- **Backend:** 20+ API endpoints, 12 миграций БД
- **База данных:** 6 основных таблиц (products, orders, users, regions, payments, events)
- **Тестирование:** Unit тесты для критичных компонентов

### Навыки, продемонстрированные в проекте

- **Frontend разработка:** React, Next.js, TypeScript, современные паттерны (hooks, context, state management)
- **Backend разработка:** Go, проектирование API, работа с БД, аутентификация
- **DevOps:** Docker, контейнеризация, настройка CI/CD
- **Базы данных:** PostgreSQL, проектирование схем, миграции, оптимизация запросов
- **Интеграции:** Работа с внешними API, webhooks, платежные системы
- **UI/UX:** Адаптивный дизайн, анимации, доступность
- **Архитектура:** Проектирование масштабируемых приложений, паттерны проектирования
