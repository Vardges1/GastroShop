## Проектный опыт

### GastroShop — E-commerce платформа (Backend на Go)
**Период:** [Указать период]  
**Роль:** Backend разработчик  
**Стек:** Go 1.22, Gin, PostgreSQL, Docker, JWT, Prometheus

**Описание проекта:**
Полнофункциональная e-commerce платформа для продажи гастрономических товаров с REST API, системой заказов, интеграцией платежных систем и AI-рекомендациями.

**Реализованный функционал:**

**Backend API (Go):**
- Разработал REST API с 20+ endpoints для работы с товарами, заказами, пользователями
- Реализовал систему аутентификации и авторизации на JWT (access + refresh tokens)
- Создал middleware для проверки токенов и прав доступа (admin/customer)
- Разработал сервисный слой с бизнес-логикой (AuthService, ProductService, OrderService, PaymentService)
- Реализовал Repository pattern для работы с базой данных
- Создал систему обработки платежей с поддержкой нескольких провайдеров (Strategy pattern)
- Реализовал обработку webhook от платежных систем с валидацией HMAC-SHA256
- Разработал AI-сервис для рекомендаций продуктов на основе rule-based алгоритмов
- Интегрировал Prometheus для сбора метрик приложения

**База данных:**
- Спроектировал схему БД с 6 основными таблицами (users, products, orders, payments, regions, events)
- Написал 12 миграций для версионирования схемы БД
- Оптимизировал запросы с использованием индексов (GIN для массивов, B-tree для поиска)
- Реализовал работу с JSONB для хранения сложных структур данных
- Использовал транзакции для обеспечения целостности данных

**Архитектура:**
- Применил трехслойную архитектуру (handlers → services → repositories)
- Использовал dependency injection для упрощения тестирования
- Реализовал паттерны проектирования: Repository, Service Layer, Strategy, Middleware
- Организовал модульную структуру кода с четким разделением ответственности

**Тестирование:**
- Написал unit-тесты для всех сервисов (AuthService, ProductService, OrderService, PaymentService)
- Создал integration-тесты для API endpoints
- Достиг покрытия кода тестами 90%+ для критических модулей
- Использовал моки для изоляции тестируемых компонентов

**DevOps:**
- Настроил Docker-контейнеры для backend и БД
- Создал Docker Compose конфигурацию для оркестрации сервисов
- Настроил Nginx как reverse proxy
- Реализовал health checks для контейнеров
- Настроил сбор метрик в Prometheus

**Ключевые достижения:**
- Разработал масштабируемую архитектуру с четким разделением слоев
- Реализовал безопасную систему аутентификации с refresh tokens
- Интегрировал платежные системы с обработкой webhook
- Обеспечил высокое покрытие тестами (90%+)
- Оптимизировал запросы к БД для работы с большими объемами данных

**Технологии:** Go 1.22, Gin Framework, PostgreSQL 15, JWT, bcrypt, Docker, Docker Compose, Nginx, Prometheus, golang-migrate, pq (PostgreSQL driver)

- Реализовал микросервисную архитектуру с разделением ответственности

- Настроил CI/CD для автоматизации тестирования и деплоя
- Интегрировал сервисы через JWT и доменные события
